djng

Kick-start your Django and AngularJS based web applications in minutes

View on GitHub Download .zip Download .tar.gz

Kick-start your Django and AngularJS based web applications in minutes. Your project will be deployed on Heroku so you can focus on building apps and not infrastructure.

What you get:

  • Django backend providing a REST API.
  • AngularJS client including Restangular and UI Router.
  • Architecture following the the twelve-factor app methodology.
  • Configured Heroku environment including logging and monitoring.
  • Everything is built on Heroku - No need to commit build artifacts to your repository.

Prerequisites

  • Installed Python and Virtualenv in a unix-style environment. See this guide for guidance.
  • An installed version of Postgres to test locally. Also create a user and database for your project.
  • Installed Node.js.
  • Installed Grunt and Bower.
  • Installed Compass.
  • Installed Heroku Toolbelt.
  • A Heroku user account. Signup is free and instant.

Bootstrap your project

mkdir myproject && cd $_
wget -qO- https://raw.githubusercontent.com/djng/djng-init/master/djng.sh | bash
